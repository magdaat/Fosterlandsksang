# Digitalisering för bevarande och tillgänglighet

Digitaliseringen av Fosterländsk sång vid Götha Canals öpnande i Hans Maj:t konungens och den kongl. famillens närvaro, den 26 september 1832, är ett projekt som har genomförts inom ramen för kursen Digitalisering för bevarande och tillgänglighet inom masterprogrammet i Biblioteks- och informationsvetenskap år 2025 vid Borås högskola.
